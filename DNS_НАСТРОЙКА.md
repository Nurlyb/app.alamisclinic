# Настройка DNS для app.alamisclinic.kz

## Проблема
Домен app.alamisclinic.kz не работает, потому что DNS запись не настроена.

## Текущая ситуация
- ✅ Приложение работает на сервере: http://localhost:3000
- ✅ Nginx настроен и работает
- ✅ Сервер доступен по IP: 185.129.49.186
- ❌ DNS запись для app.alamisclinic.kz отсутствует

## Что нужно сделать

### 1. Зайти в панель управления DNS
Зайдите в панель управления доменом alamisclinic.kz (обычно это регистратор домена или хостинг-провайдер).

### 2. Добавить A-запись
Создайте новую DNS запись со следующими параметрами:

```
Тип записи: A
Имя (Name): app
Значение (Value/Points to): 185.129.49.186
TTL: 3600 (или оставьте по умолчанию)
```

### 3. Подождать распространения DNS
DNS записи обновляются не мгновенно. Обычно это занимает:
- 5-15 минут (если TTL маленький)
- До 24-48 часов (в редких случаях)

### 4. Проверить DNS
После добавления записи проверьте её командой:
```bash
nslookup app.alamisclinic.kz 8.8.8.8
```

Должно вернуть:
```
Name:   app.alamisclinic.kz
Address: 185.129.49.186
```

### 5. Установить SSL сертификат
Когда DNS заработает, выполните на сервере:
```bash
ssh root@185.129.49.186
certbot --nginx -d app.alamisclinic.kz --non-interactive --agree-tos --email admin@alamisclinic.kz
systemctl reload nginx
```

## Альтернатива: Временный доступ по IP

Пока DNS не настроен, можно проверить работу приложения напрямую по IP:

```
http://185.129.49.186:3000
```

Но это не рекомендуется для продакшена, так как:
- Нет SSL (небезопасно)
- Порт 3000 может быть закрыт файрволом
- Неудобно для пользователей

## Где настраивать DNS?

Основной домен alamisclinic.kz указывает на IP:
- 63.176.8.218
- 35.157.26.135

Это значит, что DNS управляется через:
- Регистратора домена (например, nic.kz, domains.kz)
- Или через хостинг-провайдера
- Или через Cloudflare/другой DNS сервис

Найдите где у вас управление DNS для домена alamisclinic.kz и добавьте там A-запись для поддомена "app".
