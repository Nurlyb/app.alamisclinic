# Исправлена форма редактирования данных пациента

## Проблема
При нажатии кнопки "Редактировать" в карточке пациента поля формы были пустыми, хотя должны были быть заполнены текущими данными пациента.

## Причина
Форма использовала `defaultValues` при инициализации, но данные пациента загружаются асинхронно через API. К моменту создания формы данные еще не были загружены, поэтому `defaultValues` были пустыми.

## Решение
Использован `useEffect` для обновления значений формы после загрузки данных пациента через метод `reset()` из react-hook-form.

## Изменения в коде

### Файл: `components/schedule/MedicalRecordForm.tsx`

**1. Добавлен импорт useEffect**:
```typescript
import { useState, useEffect } from 'react';
```

**2. Добавлен метод reset в деструктуризацию useForm**:
```typescript
const {
  register: registerPatient,
  handleSubmit: handleSubmitPatient,
  formState: { errors: patientErrors },
  setValue: setPatientValue,
  watch: watchPatient,
  reset: resetPatient, // Добавлено
} = useForm<PatientEditFormData>({
  resolver: zodResolver(patientEditSchema),
});
```

**3. Заменен useState на useEffect для обновления формы**:

**Было**:
```typescript
const {
  // ...
} = useForm<PatientEditFormData>({
  resolver: zodResolver(patientEditSchema),
  defaultValues: {
    fullName: patientDetails?.fullName || '',
    iin: patientDetails?.iin || '',
    // ...
  },
});

useState(() => {
  if (patientDetails) {
    setPatientValue('fullName', patientDetails.fullName);
    // ...
  }
});
```

**Стало**:
```typescript
const {
  // ...
} = useForm<PatientEditFormData>({
  resolver: zodResolver(patientEditSchema),
});

useEffect(() => {
  if (patientDetails) {
    resetPatient({
      fullName: patientDetails.fullName,
      iin: patientDetails.iin || '',
      dob: patientDetails.dob || '',
      gender: (patientDetails.gender as 'MALE' | 'FEMALE') || undefined,
      phone: patientDetails.phone,
      address: patientDetails.address || '',
    });
  }
}, [patientDetails, resetPatient]);
```

## Как работает

1. Компонент монтируется, форма создается с пустыми значениями
2. Запускается запрос на загрузку данных пациента (`useQuery`)
3. Когда данные загружаются, `patientDetails` обновляется
4. `useEffect` срабатывает на изменение `patientDetails`
5. Вызывается `resetPatient()` с актуальными данными
6. Форма заполняется текущими данными пациента

## Преимущества решения

1. **Корректная работа**: Форма всегда заполняется актуальными данными
2. **Реактивность**: При изменении данных пациента форма автоматически обновляется
3. **Чистый код**: Использование стандартного метода `reset()` из react-hook-form
4. **Производительность**: Обновление происходит только при изменении данных
