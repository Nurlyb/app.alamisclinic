# Исправления ошибок в объединенном расписании

## Исправленные ошибки

### ❌ TypeError: operations.reduce is not a function

**Проблема:** 
- Ошибка возникала в строке 228 файла `app/schedule/page.tsx`
- Пытались вызвать `operations.reduce()`, но `operations` мог быть `undefined` или не массивом

**Решение:**
```typescript
// Было:
const operationsByTime = operations.reduce((acc, op: Operation) => {
  // ...
}, {} as Record<string, Operation[]>);

// Стало:
const operationsByTime = (operations || []).reduce((acc, op: Operation) => {
  // ...
}, {} as Record<string, Operation[]>);
```
