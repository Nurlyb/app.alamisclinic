# Команды для исправления ошибок на сервере

## Выполни эти команды на сервере:

```bash
cd /var/www/app.alamisclinic && git pull origin main && npm run build && pm2 restart alamis-clinic && sleep 5 && pm2 logs alamis-clinic --lines 30 --nostream
```

## Что делают команды:
1. `cd /var/www/app.alamisclinic` - переходим в папку приложения
2. `git pull origin main` - получаем последние изменения (убрали output: 'standalone')
3. `npm run build` - пересобираем приложение без standalone режима
4. `pm2 restart alamis-clinic` - перезапускаем приложение
5. `sleep 5` - ждем 5 секунд
6. `pm2 logs alamis-clinic --lines 30 --nostream` - проверяем логи

## После выполнения проверь:
```bash
curl http://localhost:3000
```

Должна вернуться HTML страница без ошибок в логах.
