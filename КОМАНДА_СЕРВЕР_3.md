# Команды для восстановления других приложений

## 1. Проверяем структуру приложений
```bash
ls -la /var/www/apps/KilemSpaPro/backend/
ls -la /var/www/api.aisell/
```

## 2. Ищем ecosystem.config.js файлы
```bash
find /var/www -name "ecosystem.config.js" -type f 2>/dev/null
```

## 3. Проверяем package.json для понимания структуры
```bash
cat /var/www/apps/KilemSpaPro/backend/package.json | grep -A 5 '"scripts"'
cat /var/www/api.aisell/package.json | grep -A 5 '"scripts"'
```

## 4. Запускаем kilemspapro-backend
```bash
cd /var/www/apps/KilemSpaPro/backend && pm2 start ecosystem.config.js 2>/dev/null || pm2 start dist/server.js --name kilemspapro-backend
```

## 5. Запускаем aisell-api
```bash
cd /var/www/api.aisell && pm2 start ecosystem.config.js 2>/dev/null || pm2 start dist/main.js --name aisell-api
```

## 6. Сохраняем конфигурацию PM2
```bash
pm2 save
```

## 7. Проверяем статус всех приложений
```bash
pm2 status
```

## 8. Проверяем работу app.kilemspa.kz
```bash
curl -I http://localhost:3001 2>/dev/null || curl -I http://localhost:3002 2>/dev/null || curl -I http://localhost:4000
```

## Выполнить все команды одной строкой:
```bash
ls -la /var/www/apps/KilemSpaPro/backend/ && ls -la /var/www/api.aisell/ && find /var/www -name "ecosystem.config.js" -type f 2>/dev/null && cd /var/www/apps/KilemSpaPro/backend && pm2 start ecosystem.config.js 2>/dev/null || pm2 start dist/server.js --name kilemspapro-backend && cd /var/www/api.aisell && pm2 start ecosystem.config.js 2>/dev/null || pm2 start dist/main.js --name aisell-api && pm2 save && pm2 status
```
