# Решение проблемы с DNS

## Проблема

DNS запись для app.alamisclinic.kz не работает уже 6+ часов. 

**Причина:** DNS серверы домена (nsone.net) не возвращают запись для поддомена app.

```bash
$ dig app.alamisclinic.kz @dns1.p01.nsone.net
# Пусто - записи нет
```

## Возможные причины

1. **Запись не синхронизировалась** - ps.kz не передал изменения на nsone.net
2. **Неправильный формат** - возможно нужно указать полное имя с точкой: `app.alamisclinic.kz.`
3. **Проблема с зоной** - нужно обновить зону DNS вручную
4. **Кеш DNS** - старые записи кешируются

## Решение 1: Проверь и пересохрани DNS запись

Зайди в панель ps.kz и проверь:

1. **Запись точно сохранена?**
   - Тип: A
   - Имя: app (или app.alamisclinic.kz)
   - Значение: 185.129.49.186
   - TTL: 3600

2. **Попробуй разные форматы имени:**
   - `app` (без домена)
   - `app.alamisclinic.kz` (с доменом)
   - `app.alamisclinic.kz.` (с точкой в конце)

3. **Есть ли кнопка "Обновить зону" или "Применить изменения"?**
   - Нажми её, если есть

4. **Проверь NS записи**
   - Убедись, что NS серверы правильные: dns1-4.p01.nsone.net

## Решение 4: Использовать Cloudflare (рекомендуется)

Если ps.kz не работает нормально, можно перенести DNS на Cloudflare:

1. Зарегистрируйся на cloudflare.com
2. Добавь домен alamisclinic.kz
3. Cloudflare даст тебе новые NS серверы
4. Измени NS серверы в ps.kz на серверы Cloudflare
5. Добавь A-запись в Cloudflare: app → 185.129.49.186
6. DNS заработает за 5-10 минут

## Что делать сейчас

### Шаг 1: Проверь запись в ps.kz
Зайди в панель и убедись, что запись точно там есть и сохранена.

### Шаг 2: Попробуй обновить зону
Найди кнопку "Обновить зону" или "Применить изменения" и нажми.

### Шаг 3: Подожди еще 2-4 часа
Иногда DNS может распространяться до 24-48 часов (хотя это редко).

### Шаг 4: Свяжись с поддержкой ps.kz
Если ничего не помогает, напиши в поддержку ps.kz и спроси почему DNS не обновляется.

### Шаг 5: Рассмотри Cloudflare
Это бесплатно и работает намного быстрее и надежнее.

## Команды для проверки

```bash
# Проверка у авторитетных DNS серверов
dig app.alamisclinic.kz @dns1.p01.nsone.net
dig app.alamisclinic.kz @dns2.p01.nsone.net

# Проверка у Google DNS
dig app.alamisclinic.kz @8.8.8.8

# Проверка у Cloudflare DNS
dig app.alamisclinic.kz @1.1.1.1

# Полная трассировка DNS
dig app.alamisclinic.kz +trace
```
